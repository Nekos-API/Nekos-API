# Get an Image by ID

This endpoint allows you to get an image by its ID.

## Endpoint Details

-   URL: `/v4/images/{id}`
-   Method: `GET`
-   Response: `JSON`

## Parameters

| Location | Name | Type    | Description    | Required |
| -------- | ---- | ------- | -------------- | -------- |
| Path     | `id` | String  | The image's ID | Yes      |

## Example Request

<Tabs items={["Python 3", "JavaScript"]}>
    <Tabs.Tab>
        ```python filename="image_details.py" showLineNumbers

        res = requests.get(f"https://api.nekosapi.com/v4/images/{image_id}")
        res.raise_for_status()

        ```
