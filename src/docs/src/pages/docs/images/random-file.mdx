# Get a Random Image File

This endpoint allows you to get a redirect to a random image's file URL, filtering by tags, characters, artists, etc.

## Endpoint Details

-   URL: `/v4/images/random`
-   Method: `GET`
-   Response: `JSON`

## Parameters

| Location | Name            | Type                                                      | Description                                                                                                    | Required |
| -------- | --------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | -------- |
| Query    | `rating`        | Array of strings [safe, suggestive, borderline, explicit] | The (age) rating of the image.                                                                                 | No       |
| Query    | `artist`        | Array of integers                                         | The artist's ID.                                                                                               | No       |
| Query    | `tags`          | Array of strings                                          | The tags names, comma-delimited.                                                                               | No       |

## Example Request

<Tabs items={["Python 3", "JavaScript"]}>
    <Tabs.Tab>
        ```python filename="random_image_file.py" showLineNumbers
        import requests

        res = requests.get("https://api.nekosapi.com/v4/images/random/file")
        res.raise_for_status()

        data = res.content
        ```
